A plugin to perform various bridge federation related activities

== Description ==

Pretty easy setup. just install and activate it on the site of your choice.

Setups a host of bridge federation of India related functions. Divided into different modules
1. Tourney Pages and Results
	a. XMLRPC bfi.postHTML - This allows posting of html contents to specific pages. Typical use is for posting results.
	b. XMLRPC bfi.getTourneys - Retrieves the list of tourney pages created so far
	c. XMLRPC bfi.addTourney - Creates a new tourney with specified pages using tourney template

== Installation ==

1. Unpack the entire contents of this plugin zip file into your `wp-content/plugins/` folder locally
2. Upload to your site
3. Navigate to `wp-admin/plugins.php` on your site (your WP plugin page)
4. Activate this plugin

OR you can just install it with WordPress by going to Plugins >> Add New >> and type this plugin's name
