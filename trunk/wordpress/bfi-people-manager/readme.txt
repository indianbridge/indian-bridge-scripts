A plugin to manage people associated with Bridge Federation of India (BFI) like executive committee, directors etc.

== Description ==

Pretty easy setup. just install and activate it on the site of your choice.

Custom Post Types for People and State Associations is created

== Installation ==

1. Unpack the entire contents of this plugin zip file into your `wp-content/plugins/` folder locally
1. Upload to your site
1. Navigate to `wp-admin/plugins.php` on your site (your WP plugin page)
1. Activate this plugin

OR you can just install it with WordPress by going to Plugins >> Add New >> and type this plugin's name
